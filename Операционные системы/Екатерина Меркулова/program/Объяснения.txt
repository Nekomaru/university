1. Переменная ProcessorTimeUnitLeft хранит кол-во оставшихся от кванта тактов на текущий момент.
Если эта переменная равна нулю, то процесс исчерпал выделенный ему квант

2. Очередь готовых процессов реализована классом TReadyQueue, который наследуется от класса TList. Класс TList представляет собой список и методы для работы с ним. В TReadyQueue реализовано два новых метода. PickProcess - для выборки очередного процесса в соответствии с заданием (максимальный приоритет), IsEmpty - для проверки списка на пустоту.

3. Функция generateRandomPrority используется для генерирования случайного приоритета новому процессу

4. Синхронизация происходит с помощью переменной IsIODeviceAvailable
